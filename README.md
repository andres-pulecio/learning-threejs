# learning-threejs

This repository saves the test projects to practice with this Three.js tool.

## Installation

It's required python installation.

```bash
sudo apt-get update
sudo apt-get install python3.6
```

## How to run?
Execute this command:
```bash
python3 -m http.server
```
Go to:
http://localhost:8000/
